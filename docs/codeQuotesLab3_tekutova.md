**Блоки кода**
Встроенный (inline) код:
Команда `git status` показывает состояние репозитория.
Многострочный код: 
``` csharp
print("Hello, Markdown")
```
Можно указать язык: python, js,bash, csharp и др.
**Цитаты**
Цитата создается символом >:
>Это цитата в Markdown
Вложенная цитата: 
Первый уровень
>>Второй уровень
---
**Пример команды Git в виде inline-кода:**
`git add .`
**Блок кода на любом языке программирования:**
```python
a=int(input())
print(a)
```
**Блок кода без указания языка:**
```
a=int(input())
print(a)
```
**Две цитаты: обычная и вложенная:**
>Обычная цитата
>>Вложенная цитата
